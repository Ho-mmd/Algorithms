# Stack

- Container adaptor with LIFO(Last In First Out) Working
- New element is added at top and an element is removed from that only

------------------------------------------------------

### Example

```
#include <stack>

std::stack<data_type> stk;

stk.push(data);

// Return stack top value
stk.top(data);

// Remove stack top value
stk.pop(data);

stk.size();

```
