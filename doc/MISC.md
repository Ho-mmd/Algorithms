### Coordinate Compression

An algorithm used to know only the relationship between numbers, regardless of the value of the number

```
// If input is [1, 50, 100, 1000, 10000]
// Output is [0, 1, 2, 3, 4]

// If input is [3, -10, 9, 3, 11]
// Output is [1, 0, 2, 1, 3]

// If input is [8, 7, 7, 8]
// Output is [1, 0, 0, 1]
