# Divide and Conquer

- Solve a problem by dividing it into smaller problems
- Steps
    - Divide : Divide problem
    - Conquer : Solve subproblem recursively
    - Combine : Solve the original problem by combining the answer to subproblem
- Usage
    - Merge sort
    - Quick sort
    - Binary search
    - FFT
